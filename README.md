# 📰 二蛋信息采集系统 (Erdan News)

**二蛋信息采集系统** 是一个轻量级、自动化且由 AI 驱动的新闻与信息聚合工具。它可以根据你设定的规则，每天定时从指定的 RSS 源抓取内容，并利用 Google Gemini 大语言模型筛选出与你追踪关键字高度相关的资讯，最终为你生成不超过100字的中文精华摘要。

## ✨ 核心特性

*   **🤖 AI 智能过滤与总结**: 接入 Gemini API，能够读懂长篇大论，自动过滤无关信息，只把最核心的内容总结给你。
*   **🌐 动态配置源与关键字**: 拥有直观的 Web 界面，可以随时修改要追踪的关键字（如："AI", "开源", "大模型"）和想要采集的 RSS 站点（如 V2EX, HackerNews），修改后立即生效。
*   **⏰ 自动化定时调度**: 后台调度器开机即忘，每天自动运行，默默为你整理当天的信息简报。
*   **🛠️ 模拟模式**: 在未配置 API Key 的情况下，系统会自动进入模拟模式，直接展示抓取的原文，方便前期调试与体验。
*   **📦 极简部署**: 使用 `uv` 作为包管理工具，一键安装依赖，配合本地 SQLite 数据库，无需繁琐的环境配置。

## 🚀 快速开始

本项目使用 `uv` 来管理虚拟环境和依赖。

### 1. 安装依赖
```bash
# 确保你已经安装了 uv (https://github.com/astral-sh/uv)
uv sync
```

### 2. 启动系统
我们提供了几个便捷的批处理脚本供 Windows 用户快速启动：

*   **`start_all.bat` (推荐)**: 一键同时启动“后台定时采集调度器”和“前台 Web 交互界面”。
*   **`debug_scraper.bat`**: 手动执行一次抓取和 AI 总结任务，常用于调试和强制刷新数据。
*   **`start_web.bat`**: 仅启动 Web 界面。
*   **`start_scheduler.bat`**: 仅启动后台定时任务。

### 3. 配置系统 (Web界面内)
1. 启动 `start_all.bat` 后，浏览器会自动打开网页（通常是 `http://localhost:8501`）。
2. 点击顶部的 **"⚙️ 系统配置"** 标签页。
3. 在 **API Key 配置** 中填入你的 `GEMINI_API_KEY`（如果不填将使用带前缀的模拟模式）。
4. 在 **追踪关键字** 和 **采集源 (RSS)** 中自定义你的偏好并保存。

## 📂 项目结构

*   `app.py`: Streamlit Web 前端交互界面。
*   `src/scraper.py`: 负责从各个 RSS 节点拉取原始文章数据的爬虫模块。
*   `src/summarizer.py`: 与 Gemini 交互的核心模块，负责判断相关性与生成摘要。
*   `src/scheduler.py`: 后台定时任务调度器。
*   `src/database.py`: SQLite 数据库结构与连接配置。
*   `src/config.py`: 用户动态配置（关键字、采集源）的读取与保存逻辑。

## 🤝 贡献与定制
如果你有特定的网站没有 RSS，可以自行修改 `src/scraper.py`，引入 `BeautifulSoup` 或 `Playwright` 编写定制化的爬虫逻辑。
